# Calculator

A simple web-based calculator built using **HTML**, **CSS**, and **Vanilla JavaScript**. It performs basic arithmetic operations (DMAS) and includes additional functions such as **Clear (C)**, **Percentage (%)**, and **Sign Change (+/-)**.

---

## Description

This project was created as a beginner-friendly collaboration to practice front-end web development fundamentals. The interface was built with HTML and CSS, while all calculator logic and interactions were implemented using JavaScript.

The project focuses on writing clean, readable code while understanding how HTML, CSS, and JavaScript work together to create an interactive web application.

## Features

- Perform basic arithmetic operations:
  - Addition
  - Subtraction
  - Multiplication
  - Division
- Percentage (%) calculation
- Sign change (+/-) functionality
- Clear (C) button to reset the calculator
- Decimal number support
- Division-by-zero error handling

## How to Run

1. Download or clone the project.
2. Extract the files if they are inside a ZIP archive.
3. Ensure the following files are in the same folder:

   ```text
   Calculator.html
   style-calculator.css
   script.js
   ```

4. Double-click **Calculator.html**

**OR**

1. Open the project folder in Visual Studio Code.
2. Open `Calculator.html`.
3. Run it using Live Server (if installed), or simply open it in your preferred web browser.

No installation, dependencies, or internet connection are required.
